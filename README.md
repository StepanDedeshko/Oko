# Zabbix Duty Panel

Версия с нормальным рабочим интерфейсом:

- сверху нет вкладок;
- справа список продуктов и дашбордов;
- в центре открывается выбранный дашборд;
- дашборд графиков показывает 8 графиков сеткой;
- страница проблем открывается отдельным экраном;
- кнопки периода показываются только на дашбордах с графиками;
- на странице проблем кнопки периода скрыты.

## Что править

Открой `config.json`.

### Zabbix

В блоке `zabbix_instances` поменяй:

```json
"base_url": "https://zabbix-product1.local",
"login_url": "https://zabbix-product1.local/index.php"
```

на свои реальные адреса.

### Графики

В блоке `products -> dashboards -> graphs` меняй ссылки:

```json
"url": "https://zabbix-product1.local/chart2.php?graphid=11111"
```

на свои ссылки графиков.

### Страница проблем

В дашборде типа `problems_page` меняй:

```json
"url": "https://zabbix-product1.local/zabbix.php?action=problem.view"
```

на свою страницу проблем с уже настроенными фильтрами.

## Сетка графиков

По умолчанию:

```json
"graph_columns": 4
```

Это значит 8 графиков будут 4 в ряд и 2 ряда.

Если хочешь 2 в ряд:

```json
"graph_columns": 2
```

## Масштаб графиков

Если графики крупные:

```json
"web_zoom_factor": 0.75
```

Если мелкие:

```json
"web_zoom_factor": 1.0
```

## Запуск

Двойной клик по:

```text
Установить Zabbix Duty Panel.desktop
```

Если не запускается, правой кнопкой по файлу и выбери “Разрешить запуск”.
