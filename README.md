# Око

Око — Linux/PySide6-приложение для дежурного, которое показывает Zabbix-графики, режим дежурства, уведомления, заметки и обновляется через GitHub Releases.

## Возможности

- просмотр Zabbix-графиков;
- продукты и страницы;
- режим дежурства;
- duty triggers;
- уведомления;
- заметки;
- темы оформления;
- обновление через GitHub Releases;
- режим разработчика.

## Требования

- Linux;
- Python 3;
- доступ к Zabbix и рабочим страницам, указанным в локальной конфигурации;
- зависимости из `requirements.txt`.

## Запуск из исходников

```bash
cd ~/Загрузки/oko_codex_project
python3 -m venv .venv
source .venv/bin/activate
python3 -m pip install --upgrade pip
python3 -m pip install -r requirements.txt
python3 main.py
```

## Установленная версия

Установленная версия запускается через созданный ярлык или рабочий скрипт запуска. Локальный файл `config.json` хранит настройки конкретного рабочего места и не должен попадать в коммиты.

## Обновление

Обновление приложения выполняется через GitHub Releases штатным механизмом проекта. Скрипты `UPDATE_OKO.sh` и `RELEASE_OKO.sh` меняются только в задачах, которые явно относятся к обновлению или релизу.

## Проверки для разработчика

```bash
python3 -m py_compile main.py
find app -name "*.py" -print0 | xargs -0 -n1 python3 -m py_compile
python3 -m unittest discover -s tests
```

## Важные правила разработки

- `config.json` нельзя коммитить;
- секреты и реальные URL нельзя коммитить;
- `UPDATE_OKO.sh` и `RELEASE_OKO.sh` трогать только если задача явно про обновление/релиз;
- после изменений запускать `py_compile` и `unittest`.
