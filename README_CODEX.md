# Как работать с проектом в Codex

1. Загрузить этот проект в GitHub/GitLab или открыть как рабочую папку.
2. Дать Codex содержимое файла `CODEX_START_PROMPT.md`.
3. Для каждой задачи использовать `TASK_TEMPLATE.md`.
4. После изменений требовать запуск проверки компиляции.
5. Не делать большие изменения одним коммитом.

## Быстрый старт локально

```bash
chmod +x run_oko.sh
./run_oko.sh
```
