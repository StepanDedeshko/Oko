ИСПРАВЛЕНИЕ БЕЛОГО ФОНА В ГРАФИКАХ

Проблема:
иногда график внутри карточки был меньше области просмотра,
и вокруг / снизу пробивался белый фон страницы Zabbix.

Что исправлено:
1. QWebEngineView для графиков теперь имеет тёмный фон.
2. QWebEnginePage тоже получает тёмный backgroundColor.
3. В JS-скрипте для fit_graphs принудительно задаётся:
   html, body { background: #0b0b0b }
4. Изображение графика центрируется и растягивается аккуратнее.
5. Для страницы Проблемы тоже включён тёмный фон webview,
   чтобы не было белых вспышек при загрузке.

Если захочешь, следующим патчем можно ещё сделать:
- цвет фона графиков под текущую тему, а не просто тёмный;
- более плотную авто-высоту карточек под реальную высоту графика.
