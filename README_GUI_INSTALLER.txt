ГРАФИЧЕСКИЙ УСТАНОВЩИК ОКО

Основной файл:
    INSTALL_OKO_GUI.sh

Он запускает:
    installer_gui.py

Если на системе нет tkinter, скрипт предложит установить:
    python3-tk

Если пользователь откажется, будет запущен обычный терминальный установщик:
    INSTALL_OKO.sh

Для Windows добавлен простой запуск:
    INSTALL_OKO_GUI.bat

Но основная целевая система проекта — Linux/Ubuntu/Cinnamon.
