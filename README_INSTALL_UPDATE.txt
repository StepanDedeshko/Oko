INSTALL_OKO.sh
Установка приложения для конечного пользователя.

UPDATE_OKO.sh
Обновление приложения из нового архива.
Сохраняет config.json.
Делает backup в папку _backups.

ROLLBACK_OKO.sh
Откат на последнюю резервную копию.

run_oko.sh
Обычный запуск приложения.

CREATE_DESKTOP_SHORTCUT.sh
Создание ярлыка в меню приложений.
