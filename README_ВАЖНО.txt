ДЕЖУРКА — ВЕРСИЯ ПОД ТВОЙ CONFIG.JSON

Я взял твои настройки из загруженного config.json и сохранил ссылки.

Главное исправление:
- FacePay -> Графики теперь строго вертикально
- Сфера -> Графики теперь строго вертикально
- один график = одна строка во всю ширину
- поле columns больше не собирает плитку 4 в ряд
- внутри config.json выставлено columns=1 у всех дашбордов

Если снова видишь 4 графика в ряд:
1. Значит запускается старая папка.
2. Удали старую папку проекта.
3. Распакуй этот архив заново.
4. Запусти ЗАПУСТИТЬ_ДЕЖУРКУ.sh.

Настройка высоты карточки графика:
config.json -> settings -> graph_card_min_height

Например:
420 — нормально
520 — крупнее
650 — еще крупнее

Запуск:
ПКМ по файлу ЗАПУСТИТЬ_ДЕЖУРКУ.sh -> Свойства -> Права -> Разрешить выполнение.
Потом двойной клик или через терминал:
./ЗАПУСТИТЬ_ДЕЖУРКУ.sh
