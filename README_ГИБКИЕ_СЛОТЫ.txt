ДЕЖУРКА — ГИБКИЕ СЛОТЫ ГРАФИКОВ

Теперь пользователям не нужно править код.

В приложении справа появился пункт:

    Настройки графиков

Что там можно делать:
1. Выбрать продукт.
2. Выбрать дашборд "Графики".
3. Включить/выключить слот.
4. Изменить название графика.
5. Вставить URL графика.
6. Добавить новый слот.
7. Сохранить config.json.

После сохранения:
- нажми кнопку "Перезагрузить дашборды"
или
- перезапусти приложение.

FacePay:
- сейчас активен только первый график;
- остальные старые пустые/тестовые слоты выключены;
- когда понадобятся новые графики, включи слот и вставь URL.

Графики всё равно идут вертикально:
один график в строку.
