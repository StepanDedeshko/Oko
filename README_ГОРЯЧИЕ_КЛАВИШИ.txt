ГОРЯЧИЕ КЛАВИШИ

В меню "Настройки" добавлен пункт:

    Горячие клавиши

Там отображается справка по комбинациям:

- F5 — обновить текущий раздел
- Ctrl + R — обновить текущий раздел
- F11 — полный экран / обычный режим
- Esc — выйти из полного экрана
- Alt + ↓ — открыть выпадающий список
- Tab / Shift + Tab — переход по элементам интерфейса

Пока это справочный экран.
Редактирование горячих клавиш можно добавить позже.
