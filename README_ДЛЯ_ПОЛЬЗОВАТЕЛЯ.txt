ОКО 0.2 [pre-release]

Установка с графическим окном:
1. Распакуйте архив.
2. Откройте папку.
3. Запустите:
   INSTALL_OKO_GUI.sh

Важно:
Установщик больше не запускает sudo и не спрашивает пароль администратора.

Если системе не хватает компонентов, установщик покажет команду, которую должен выполнить администратор, например:
   sudo apt install ...

Если графический установщик не открылся:
- будет предложен обычный терминальный установщик INSTALL_OKO.sh;
- он тоже не запускает sudo.

Путь установки по умолчанию:
   ~/Applications/Oko

Запуск после установки:
- из меню приложений: Око;
- файлом run_oko.sh;
- файлом ЗАПУСТИТЬ_ОКО.sh.

Обновление:
1. Скачайте новый архив сборки Око.
2. В установленной папке запустите UPDATE_OKO.sh.
3. Укажите путь к новому архиву .zip или .tar.gz.
4. Скрипт сохранит config.json и сделает резервную копию предыдущей версии.

Откат:
Если после обновления что-то сломалось:
1. Запустите ROLLBACK_OKO.sh.
2. Подтвердите откат.
3. Будет восстановлена последняя резервная копия.
