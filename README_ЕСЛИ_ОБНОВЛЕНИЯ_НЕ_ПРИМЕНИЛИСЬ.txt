ЕСЛИ ОБНОВЛЕНИЯ НЕ ПРИМЕНИЛИСЬ

Причина:
обновлятор сохраняет твой config.json, чтобы не потерять ссылки Zabbix.
Из-за этого новые ключи настроек могли не попасть в рабочий конфиг.

Исправлено в этой версии:
1. При запуске приложение само добавляет недостающие настройки в config.json.
2. При обновлении config.json теперь мигрируется.
3. Ссылки пользователя не затираются.
4. Убраны старые блоки otrs и fp_connect, если они остались.
5. Добавляется loading_screen, problem_counter, theme и другие новые параметры.

Если уже обновился, но изменения не появились:
запусти файл:

    ПОЧИНИТЬ_CONFIG_ПОСЛЕ_ОБНОВЛЕНИЯ.sh

Он добавит недостающие настройки в текущий config.json и не тронет твои ссылки.
