ИЗМЕНЕНА ЛОГИКА ЗАГРУЗКИ

Теперь:
1. Заставка НЕ появляется после кнопки "Войти".
2. Заставка появляется только когда пользователь выбирает раздел "Проблемы".
3. Заставка модальная — пока она открыта, нельзя кликать меню приложения.
4. При открытии "Проблемы" страница перезагружается.
5. Каждые 5 секунд во время заставки приложение пытается найти счётчик .table-stats.
6. Если число найдено — заставка закрывается.
7. Если за 30 секунд не найдено — заставка закрывается по таймауту.

ОБНОВЛЕНИЯ И МЕДИА

Обновления не должны трогать:
assets/loading

То есть твои видео и музыка должны сохраняться при обновлении.

Настройки в config.json:

"loading_screen": {
  "enabled": true,
  "show_after_login": false,
  "show_until_problem_counter_ready": false,
  "show_on_problems_open": true,
  "max_wait_ms": 30000,
  "problem_counter_poll_interval_ms": 5000
}
