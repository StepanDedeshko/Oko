ДЕЖУРКА — КАК ЗАПУСТИТЬ НА UBUNTU

Главный файл запуска:

    ЗАПУСТИТЬ_ДЕЖУРКУ.sh

Это аналог .bat для Ubuntu.

ВАЖНО:
В этой версии 1 график в ряд.
В config.json выставлено:
    settings -> graph_columns = 1
и внутри каждого dashboard:
    columns = 1

Если у тебя всё ещё 4 графика в ряд, значит запускается старая папка проекта.
Удали старую папку и распакуй этот архив заново.

Как запустить:

Вариант 1:
1. Правой кнопкой по файлу ЗАПУСТИТЬ_ДЕЖУРКУ.sh
2. Свойства
3. Права
4. Включить: Разрешить выполнение файла как программы
5. Двойной клик по файлу
6. Выбрать: Запустить

Вариант 2:
1. Открой папку проекта
2. ПКМ по пустому месту
3. Открыть в терминале
4. Вставь:
    ./ЗАПУСТИТЬ_ДЕЖУРКУ.sh

После первого запуска появится ярлык в меню приложений:
    Дежурка

Файл настроек:
    config.json
