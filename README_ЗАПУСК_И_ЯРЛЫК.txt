ДЕЖУРКА — ГОТОВАЯ ВЕРСИЯ С УСТАНОВЩИКОМ И ЯРЛЫКОМ

Что внутри:
1. Твои ссылки из config.json сохранены.
2. FacePay:
   - Графики
   - Проблемы
   - Проблемы 2
3. Сфера:
   - Графики
4. Графики идут строго вертикально: один график в строку.
5. Есть сохранение пароля локально.
6. Есть иконка приложения.
7. После первого запуска создается ярлык в меню Ubuntu: Дежурка.

Главный файл запуска:

    ЗАПУСТИТЬ_ДЕЖУРКУ.sh

Как запустить:
1. Распакуй архив.
2. Открой папку.
3. ПКМ по файлу ЗАПУСТИТЬ_ДЕЖУРКУ.sh.
4. Свойства -> Права.
5. Включи: Разрешить выполнение файла как программы.
6. Двойной клик по ЗАПУСТИТЬ_ДЕЖУРКУ.sh.

Если двойной клик снова открывает текст:
1. ПКМ по пустому месту в папке.
2. Открыть в терминале.
3. Вставить:

    ./ЗАПУСТИТЬ_ДЕЖУРКУ.sh

После первого запуска ярлык появится здесь:
    ~/.local/share/applications/dezhurka.desktop

Иконка копируется сюда:
    ~/.local/share/icons/hicolor/256x256/apps/dezhurka.png

Чтобы добавить ссылку во вторую страницу FacePay:
1. Открой config.json.
2. Найди:
    "name": "Проблемы 2"
3. Вставь свою ссылку в поле:
    "url": "..."

Если вдруг опять видишь 4 графика в ряд:
значит запускается старая папка. Удали старую папку и распакуй этот архив заново.
