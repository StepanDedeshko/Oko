ОТДЕЛЬНАЯ PORTABLE-СБОРКА ДЕЖУРКИ

Эта сборка не ставится в систему и не трогает установленную Дежурку.

Запуск:
    ./run_terminal.sh

Что внутри:
- FacePay -> Сработки с режимами Режим 1 / Режим 2;
- Дежурство -> Режим дежурства;
- настройка графиков для дежурной проверки внутри страницы режима;
- запуск через локальное .venv в этой папке.

Если зависимости сломались:
    ./reset_venv.sh
    ./run_terminal.sh
