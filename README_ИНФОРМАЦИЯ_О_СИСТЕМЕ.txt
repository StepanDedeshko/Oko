ИНФОРМАЦИЯ О СИСТЕМЕ

В меню "Настройки" добавлен пункт:

    Информация о системе

Там отображается:
- версия Ubuntu / Linux;
- ядро;
- архитектура;
- hostname;
- CPU;
- количество ядер и потоков;
- загрузка CPU;
- температура CPU, если датчик доступен;
- RAM и Swap;
- диски;
- сетевые интерфейсы и IP;
- видеокарта, если доступен lspci;
- разрешение экрана;
- версия Python, PySide6 и psutil;
- папка приложения Дежурка.

Добавлены кнопки:
- Обновить информацию
- Копировать

Кнопка "Копировать" кладёт весь отчёт в буфер обмена,
чтобы пользователь мог отправить его в Telegram/чат.
