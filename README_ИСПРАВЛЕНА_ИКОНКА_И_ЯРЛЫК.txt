ИСПРАВЛЕНА ИКОНКА И АКТИВНЫЙ ЯРЛЫК

Что исправлено:
1. Иконка теперь ставится не только на ярлык, но и на само Qt-приложение.
   Поэтому при запуске не должна появляться шестеренка.
2. Ярлык создается через безопасный launcher:
   ~/.local/bin/dezhurka-launcher.sh
3. Ярлык на рабочем столе помечается как доверенный через gio metadata::trusted.
4. Ярлык обновляется при каждом запуске приложения.

Если ярлык на рабочем столе всё равно не активный:
ПКМ по ярлыку -> Разрешить запуск.

Если после обновления всё ещё видна шестеренка:
1. Закрой Дежурку.
2. Удали старый ярлык с рабочего стола.
3. Запусти:
   ./СОЗДАТЬ_ЯРЛЫК.sh
4. Запусти новый ярлык.
