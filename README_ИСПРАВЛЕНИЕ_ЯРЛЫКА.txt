ИСПРАВЛЕНИЕ ЯРЛЫКА "ДЕЖУРКА"

Если ярлык на рабочем столе пишет, что имеет ошибки, запусти:

    СОЗДАТЬ_ЯРЛЫК.sh

Эта версия создает не прямой Exec на файл с кириллицей,
а отдельный безопасный launcher:

    ~/.local/bin/dezhurka-launcher.sh

А ярлык запускает уже его.

Как запустить:
1. ПКМ по СОЗДАТЬ_ЯРЛЫК.sh
2. Свойства -> Права
3. Разрешить выполнение файла как программы
4. Двойной клик -> Запустить

Если опять открывает текст:
1. Открой папку проекта в терминале
2. Выполни:
    ./СОЗДАТЬ_ЯРЛЫК.sh

После этого удали старый битый ярлык с рабочего стола, если он остался,
и используй новый "Дежурка.desktop".
