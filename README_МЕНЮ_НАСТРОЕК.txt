Кнопка "Обновить":
обновляет текущий открытый раздел.
Если открыт раздел "Графики" — обновляются графики этого раздела.
Если открыта страница проблем/дашборд — обновляется эта страница.

В меню "Настройки" перенесено:
- Настройки графиков
- Обновить приложение
- Пересоздать ярлык
- Удалить приложение

Также оставлены заготовки на будущее:
- Тема оформления — позже
- Профили продуктов — позже

Кнопки "Обновить всё" и "Обновить приложение" больше не висят рядом,
чтобы пользователи не путались.
