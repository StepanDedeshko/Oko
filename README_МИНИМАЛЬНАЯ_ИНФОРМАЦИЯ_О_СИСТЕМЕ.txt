МИНИМАЛЬНАЯ ИНФОРМАЦИЯ О СИСТЕМЕ

Раздел "Информация о системе" теперь показывает только:

- ОС
- CPU
- Память всего/занято
- Видеокарта

Пример:

ОС: Ubuntu 24.04.2 LTS
CPU: Intel Core i5-...
Память всего/занято: 15.6 ГБ / 6.2 ГБ (40%)
Видеокарта: Intel UHD Graphics

Кнопка "Копировать" копирует этот короткий текст в буфер обмена.
