НИЖНЯЯ HUD-ПАНЕЛЬ

Добавлена нижняя панель как в sci-fi макете.

Показывает:
1. Время последнего обновления данных.
2. Использование памяти устройства.
3. Скорость сети:
   входящая / исходящая.
4. Температуру CPU, если Ubuntu отдаёт датчики.

Для сбора показателей добавлена библиотека:

    psutil

Температура CPU:
- сначала берется через psutil.sensors_temperatures();
- если не получилось, пробует /sys/class/thermal;
- если датчик недоступен, будет показано "н/д".

Обновление панели происходит каждые 2 секунды.
