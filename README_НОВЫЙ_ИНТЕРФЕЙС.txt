НОВЫЙ ИНТЕРФЕЙС ДЕЖУРКИ

Переделано:
1. Боковая шторка убрана.
2. Список продуктов теперь сверху в выпадающем списке.
3. Разделы продукта теперь тоже сверху в выпадающем списке.
4. Периоды времени теперь в отдельном выпадающем списке.
5. Период показывается только на разделах с графиками.
6. На страницах проблем/дашбордов период скрыт.

Логика работы:
1. Вверху выбрать продукт:
   - FacePay
   - Сфера

2. Рядом выбрать раздел:
   - Графики
   - Проблемы
   - Проблемы 2

3. Если открыт раздел "Графики", рядом появится:
   - Период: 30 минут / 1 час / 3 часа / ...

4. Если открыт раздел "Проблемы" или "Проблемы 2",
   период скрывается, потому что там свои фильтры/страницы Zabbix.

Кнопка "Настройки графиков" осталась сверху.
