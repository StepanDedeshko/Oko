ОБНОВЛЕНИЕ ДЕЖУРКИ

Добавлена кнопка в верхней панели:

    Обновить приложение

Как работает:
1. Нажимаешь "Обновить приложение".
2. Выбираешь ZIP с новой версией Дежурки.
3. Приложение сохраняет текущий config.json.
4. Накатывает новые файлы.
5. Возвращает твой config.json.
6. Пересоздает ярлык, если есть СОЗДАТЬ_ЯРЛЫК.sh.
7. Просит перезапустить приложение.

Что НЕ теряется:
- config.json с твоими ссылками;
- сохраненные пароли, потому что они лежат в ~/.config/zabbix_duty_panel;
- .venv не трогается.

Также добавлен отдельный скрипт:

    ОБНОВИТЬ_ДЕЖУРКУ.sh

Им можно воспользоваться, если приложение не запускается, но нужно накатить обновление.

ВАЖНО:
ZIP обновления должен быть архивом проекта Дежурки, где есть main.py и папка app.
