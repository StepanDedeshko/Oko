ШАБЛОНЫ ПРОБЛЕМ — УПРОЩЕННАЯ ЛОГИКА

Сделано так, как просил:

1. Справа открываешь "Проблемы".
2. Сразу открывается сохраненная страница.
3. Сверху есть выпадающий список шаблонов.
4. Выбираешь шаблон и жмешь "Применить шаблон".
5. Никакого автоматического сохранения ссылки из открытой страницы больше нет.

Где добавлять шаблоны:

    Настройки проблем

Там можно:
- выбрать продукт;
- выбрать страницу проблем;
- добавить шаблон;
- указать название;
- вставить ссылку из Zabbix;
- сохранить.

Пример шаблонов:
- Критичные
- Неподтвержденные
- FacePay API
- Ошибки за сутки

В config.json это хранится внутри нужной страницы problems_page:

"url_templates": [
  {
    "title": "Критичные",
    "url": "http://..."
  }
]

Важно:
Страница проблем открывает первый шаблон как основной.
Если нужно поменять основной — поставь нужный шаблон первым в настройках.
