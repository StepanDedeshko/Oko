ПРОСТАЯ ИНФОРМАЦИЯ О СИСТЕМЕ

Раздел "Информация о системе" упрощён.

Теперь показывает только:
- ОС;
- ядро;
- архитектура;
- CPU;
- ядра/потоки;
- температура CPU, если доступна;
- общий объём RAM;
- сколько RAM используется;
- видеокарта, если удалось определить;
- разрешение экрана.

Кнопки:
- Обновить
- Копировать

Кнопка "Копировать" кладёт короткий отчёт в буфер обмена,
чтобы его можно было отправить в Telegram.
