СЧЕТЧИК ПРОБЛЕМ — ЖЕСТКОЕ ИСПРАВЛЕНИЕ

Для твоего HTML:

<div class="table-stats">Отображено с 1 до 100 из 294 найденных</div>

Теперь в config.json стоит:

"selector": ".table-stats",
"regex": "из\\s+(\\d+)\\s+найден"

Также сделано:
1. Кнопка "Обновить счётчик" на странице Проблемы.
2. Повторные попытки после загрузки страницы.
3. Если selector не сработал, приложение всё равно ищет .table-stats.
4. Если и это не сработало, ищет строку "Отображено с ... из ... найденных" по всему body.

Если всё равно не показывает:
1. Открой Проблемы в приложении.
2. Подожди 5-10 секунд.
3. Нажми "Обновить счётчик".
4. Наведи мышь на "Всего проблем: н/д" — там будет tooltip с источником/частью текста.
