СЧЕТЧИК ПРОБЛЕМ ZABBIX

Добавлен каркас счётчика для страниц типа problems_page.

На странице "Проблемы" сверху справа появится плашка:

    Всего проблем: н/д

или:

    Всего проблем: 15

Где настраивать:
config.json -> нужная страница Проблемы -> problem_counter

Пример:

"problem_counter": {
  "enabled": true,
  "title": "Всего проблем",
  "selector": "",
  "regex": "(\\d+)",
  "refresh_after_load_ms": 1200
}

Как использовать правильно:
1. Открой страницу Zabbix "Проблемы" в обычном браузере.
2. Найди место, где отображается количество проблем.
3. ПКМ по числу -> Просмотреть код / Inspect.
4. Найди CSS-селектор элемента.
5. Вставь его в поле selector.

Пример:

"selector": ".table-stats"

или:

"selector": "#problem_count"

Если в элементе текст вида:
    Всего проблем: 27

то regex можно оставить:
    "(\\d+)"

Если текст сложнее, можно указать свою регулярку.

Важно:
Если selector пустой, приложение попробует найти число само по типовым местам.
Но надежнее указать selector вручную.
