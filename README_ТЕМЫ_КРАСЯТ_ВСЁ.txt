ИСПРАВЛЕНО ПРИМЕНЕНИЕ ТЕМ

Проблема:
часть интерфейса красилась темой, а часть оставалась старой,
потому что в коде были жёстко прописанные стили.

Исправлено:
1. Темы теперь применяют не только stylesheet, но и QPalette.
2. После смены темы приложение переполирует все уже созданные виджеты.
3. Убраны жёсткие стили с карточек графиков.
4. Заголовок "Дежурка" теперь тоже красится темой.
5. Карточки, кнопки, выпадающие списки, поля, меню, таблицы,
   нижняя HUD-панель и плашка счётчика теперь завязаны на выбранную тему.

Темы:
- Simple Dark
- Mass Effect HUD
- Cerberus Red
- Matrix Green
- Omega Purple
- Amber Ops
