ТЕМЫ ОФОРМЛЕНИЯ

В Дежурку добавлены несколько тем оформления.

Теперь в меню:
    Настройки -> Тема оформления
можно выбрать одну из тем:

- Mass Effect HUD
- Cerberus Red
- Matrix Green
- Omega Purple
- Amber Ops

Выбранная тема сразу применяется к интерфейсу и сохраняется в config.json.

Ключ в config.json:
    settings.theme

Например:
    "theme": "matrix_green"
