УДАЛЕНИЕ ДЕЖУРКИ

Для удаления запусти файл:

    УДАЛИТЬ_ДЕЖУРКУ.sh

Что он удаляет:
1. Ярлык из меню Ubuntu:
   ~/.local/share/applications/dezhurka.desktop

2. Ярлык с рабочего стола:
   ~/Рабочий стол/Дежурка.desktop
   или
   ~/Desktop/Дежурка.desktop

3. Launcher:
   ~/.local/bin/dezhurka-launcher.sh

4. Иконку:
   ~/.local/share/icons/hicolor/256x256/apps/dezhurka.png

5. Профили браузера/кэш:
   ~/.zabbix_duty_panel

6. По желанию:
   сохраненные логины/пароли:
   ~/.config/zabbix_duty_panel

7. По желанию:
   саму папку приложения.

Как запустить:
1. ПКМ по УДАЛИТЬ_ДЕЖУРКУ.sh
2. Свойства -> Права
3. Разрешить выполнение файла как программы
4. Двойной клик -> Запустить

Если двойной клик открывает текст:
открой папку в терминале и выполни:

    ./УДАЛИТЬ_ДЕЖУРКУ.sh
