ДЕЖУРКА — ФИНАЛЬНАЯ ГОТОВАЯ ВЕРСИЯ

В этой сборке:
1. Нет FP_Connect.
2. Нет запуска сторонних .py приложений.
3. Нет шаблонов проблем.
4. Есть иконка приложения.
5. Есть нормальный ярлык Ubuntu.
6. Есть сохранение пароля локально.
7. Есть настройки графиков без правки кода.
8. Графики идут вертикально: один график в строку.
9. Есть простое обновление приложения.

ПЕРВЫЙ ЗАПУСК

Главный файл:

    УСТАНОВИТЬ_ДЕЖУРКУ.sh

Если двойной клик открывает текст:
1. ПКМ по пустому месту в папке.
2. Открыть в терминале.
3. Выполнить:

    ./УСТАНОВИТЬ_ДЕЖУРКУ.sh

После первого запуска появится ярлык в меню Ubuntu:

    Дежурка

Если переносишь папку в другое место, запусти:

    ./СОЗДАТЬ_ЯРЛЫК.sh

ОБНОВЛЕНИЕ ДЛЯ НЕОПЫТНОГО ПОЛЬЗОВАТЕЛЯ

Вариант 1 — через приложение:
1. Получить новый ZIP обновления.
2. Переименовать его в:

    update.zip

3. Положить в папку:

    updates

4. В приложении нажать:

    Обновить приложение

5. Согласиться.
6. Закрыть и снова открыть Дежурку.

Вариант 2 — если приложение не запускается:
1. Положить update.zip в папку updates.
2. Запустить:

    ОБНОВИТЬ_ДЕЖУРКУ.sh

Что сохраняется при обновлении:
- config.json;
- все ссылки на Zabbix;
- настройки графиков;
- сохранённые пароли;
- .venv.

Что обновляется:
- код приложения;
- иконки;
- скрипты запуска;
- ярлык.


УДАЛЕНИЕ

Для удаления приложения добавлен файл:

    УДАЛИТЬ_ДЕЖУРКУ.sh

Он удаляет ярлыки, launcher, иконку, профили браузера/кэш и по желанию сохраненные пароли и саму папку приложения.
