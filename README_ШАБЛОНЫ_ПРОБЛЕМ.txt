ШАБЛОНЫ ССЫЛОК ДЛЯ ПРОБЛЕМ ZABBIX

Добавлено для страниц типа:

    "type": "problems_page"

На странице проблем теперь есть:
1. Выпадающий список "Шаблон проблем"
2. Кнопка "Применить шаблон"
3. Кнопка "Сохранить текущую ссылку как шаблон"
4. Кнопка "Удалить шаблон"
5. Кнопка "Открыть в браузере"

Как пользоваться:
1. Открой пункт "Проблемы" или "Проблемы 2".
2. В самом Zabbix выставь фильтры как нужно.
3. Когда в адресной строке внутри приложения будет нужная ссылка,
   нажми "Сохранить текущую ссылку как шаблон".
4. Введи название, например:
   - Все критичные
   - Только FacePay API
   - Неподтвержденные
   - За последние сутки
5. Потом можно быстро выбирать этот шаблон из списка.

В config.json у страницы проблем появится блок:

    "url_templates": [
      {
        "title": "Все критичные",
        "url": "http://..."
      }
    ]

Кнопки времени "30 минут / 1 час / 3 часа" на страницах проблем не применяются.
Там используются фильтры самой страницы Zabbix.
