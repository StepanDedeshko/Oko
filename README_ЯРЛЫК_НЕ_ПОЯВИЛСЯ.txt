ЕСЛИ ЯРЛЫК 'ДЕЖУРКА' НЕ ПОЯВИЛСЯ

Запусти отдельный файл:

    СОЗДАТЬ_ЯРЛЫК.sh

Как запустить:
1. ПКМ по СОЗДАТЬ_ЯРЛЫК.sh
2. Свойства
3. Права
4. Разрешить выполнение файла как программы
5. Двойной клик -> Запустить

Если двойной клик опять открывает текст:
1. Открой папку проекта
2. ПКМ по пустому месту
3. Открыть в терминале
4. Вставь:

    ./СОЗДАТЬ_ЯРЛЫК.sh

После этого ярлык должен появиться:
1. В меню приложений Ubuntu как "Дежурка"
2. На рабочем столе как "Дежурка.desktop"

Если ярлык на рабочем столе серый:
ПКМ по нему -> Разрешить запуск.

Если в меню всё равно не видно:
выйди из учетной записи Ubuntu и войди обратно.
